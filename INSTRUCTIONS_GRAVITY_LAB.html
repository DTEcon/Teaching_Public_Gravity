Lecture 7 — Student Quick Guide
Open in Google Colab
Objective: Run and interpret gravity regressions by changing the model specification, fixed effects, covariates, and country sample — all inside your browser, with no installation required.

1 How to Run

  1. Open the Colab notebook: colab.research.google.com/…/gravity_student_lab_colab_minimal.ipynb
  2. In the Gravity Spec Controls panel, choose a preset_spec (a pre-specified option, or Custom).
  3. Click Runtime → Run all.
  4. Read the output table at the bottom of the page.
  5. Optional: Click Show code to view the underlying Python code.
  6. Optional: File → Save a copy in Drive to keep your own settings and results.

2 Available Preset Specifications

Naive GDP (OLS) Uses GDP controls without fixed effects.
Cost-Proxy (OLS) GDP controls plus additional gravity covariates (no fixed effects).
Canonical FE (OLS) Exporter + importer fixed effects, log-linear OLS. Reproduces Column (1) on Lecture 7, slide 26 (with country filter set to ALL).
Canonical FE (PPML) Exporter + importer fixed effects, PPML in levels. Reproduces Column (2) on Lecture 7, slide 26 (with country filter set to ALL). Runtime may be slightly longer.
Custom Choose the estimator, fixed effects, covariates, and country filters yourself.
In non-Custom presets the estimator, fixed effects, and covariates are set automatically; only the exporter/importer country filters remain adjustable.

Covariate definitions are documented in the CEPII gravity codebook (PDF).

3 Interpretation Guide

Variable TypeHow to Interpret the Coefficient
ln_dist (log of population-weighted distance) The coefficient is an elasticity: a 1% increase in distance changes trade by β%. Typically negative.
contig, comlang_off, rta (dummy variables) Approximate effect: 100·κ%. Exact multiplicative effect: exp(κ), so the exact percent change is 100·(exp(κ)−1)%.
Example: κ = 0.50 → approx. 50%; exact = 100·(e0.50−1) ≈ 64.9%.
Fixed Effects
With include_fe = True, exporter and importer fixed effects absorb all country-specific factors (such as the importer's or exporter's GDP and population), consistent with the multilateral-resistance logic from Lecture 7.
OLS vs. PPML
OLS uses log trade flows and therefore drops zero-flow observations. PPML estimates in levels and retains zero flows.

4 Suggested Experiments

  • Compare Naive GDP (OLS) with Canonical FE (OLS) — observe how fixed effects change the coefficients.
  • Keep fixed effects on and add or drop one bilateral covariate at a time to see how estimates shift.
  • Restrict exporters or importers to a few ISO3 codes (e.g., CHE, DEU, USA) and compare the resulting coefficients to the full sample.
